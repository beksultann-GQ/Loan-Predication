# Loan-Predication
(Description eng v)
This project is dedicated to the task of predicting loan approval based on data about bank customers. Using machine learning methods, characteristics such as the applicant's income, marital status, credit history, and other parameters are analyzed.
The aim of the project is to build a model that can automatically predict the probability of loan approval and help in decision—making in the banking system.

(Сипаттамасы қаз н)
Бұл жоба Банктің клиенттері туралы мәліметтер негізінде несиені мақұлдауды болжау міндетіне арналған. Машиналық оқыту әдістерін қолдана отырып, өтініш берушінің кірісі, отбасылық жағдайы, несиелік тарихы және басқа параметрлер сияқты сипаттамалар талданады. 
Жобаның мақсаты-несиені мақұлдау ықтималдығын автоматты түрде болжай алатын және банк жүйесінде шешім қабылдауға көмектесетін модель құру.

(описание рус в)
Данный проект посвящён задаче предсказания одобрения кредита на основе данных о клиентах банка. Используя методы машинного обучения, анализируются такие характеристики, как доход заявителя, семейное положение, кредитная история и другие параметры.
Цель проекта построить модель, которая может автоматически прогнозировать вероятность одобрения кредита и помочь в принятии решений в банковской системе.
